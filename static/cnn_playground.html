ConvNets in Action

Draw a digit (0–9) on the canvas and hit Run to watch a convolutional neural network classify it in real time. The animation shows the actual intermediate feature maps as the image passes through each layer. Hover over any layer in the visualization to inspect individual channel activations and filter weights.

The model is a small CNN (~6.2k parameters) with three 3×3 conv layers (1→8→16→32 channels), max pooling, global average pooling, and a linear classifier. It was trained for 25 epochs with the Adam optimizer on the MNIST training set (60k images) in ~17 minutes on a laptop CPU, reaching 96.3% test accuracy (i.e. this is not a state of the art model). All weights are hardcoded directly into the page source—no server or model file download—and the forward pass is pure JavaScript.

Challenge: Can you break it, and can you figure out what part of it broke?
(Hint: some of the sample images are incorrectly classified.)
